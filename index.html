HTML - The Foundation of the Web

Posted by: Abner Olivarez

HTML (Hypertext Mark Up Language is the most basic building block of the web. It defines the meaning and structure of the web content. Other technologies besides HTML are generally used to describe a web page's appearance/presentation(CSS) or functionality/behavior(Javascript).

"Hypertext" refers to links that connect web pages to one another, either within a single website or between websites. Links are a fundamental aspect of the Web. By uploading content to the Internet and linking it to pages created by other people, you become an active participation in the World Wide Web.

HTML stands for Hyper Text Mark Language. It is a markup language that defines the structure of your content. HTML consists of a series of elements, which you use to enclose, or wrap, different parts of the content to amke it appear a certain way, or act a certain way. The enclosing tags can make a word or image hyperlink to somewhere else, can italicize words, can make the font bigger or smaller,and so on.

HTML consist of elements that describe different types of content:paragraph, link, heading, images, video, etc. Web browser understand HTML and render HTML code as websites.

The Main Parts of Elements are as Follows

  1. The opening tag
  2. The closing tag
  3. The content tag
  4. The element tag
